# lizhi-mp3
李志所有专辑（除live外）的 320K mp3 整理。

## 下载
``` bash
# 下载所有专辑曲目
curl https://testingcf.jsdelivr.net/gh/lvboda/lizhi-mp3@main/install.sh | sh
```
